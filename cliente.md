**Cargas Automáticas - Guía de Configuración**

Documentación para asignar cargas automáticas de datos por cadena.

Requisitos Previos

✅ Cuenta con permisos de administrador

✅ Credenciales de la cadena configuradas (usuario/contraseña)

✅ Cadena activa en el sistema


Configuración Rápida
1. Acceder al módulo
   
Iniciar sesión → Home → Cadenas → Credenciales

2. Seleccionar credencial

Busca y selecciona el usuario al que asignarás las cargas.

3. Marcar procesos

En la sección "Procesos Adicionales", marca los procesos que necesites:


| Procesos | Frecuencia |
|---------:|------------|
|     Sell Out    |  Diaria          |      
|     Pedidos     |  Lunes          |       
|     Comparativo |   Lunes         |    
|     Forecast    |  Lunes          |  
|     Devoluciones    |  Lunes      |       
|     Compras    |  Lunes          |   
|     RecShip    |  Lunes          |   


4. Guardar

Haz clic en Guardar para aplicar los cambios.

**Frecuencia de Ejecución**

Lunes: Todos los procesos marcados

Martes - Domingo: Solo Sell Out (si está marcado)


Rango de dias a descargar: Últimos 7 días (desde hace 7 días hasta ayer)
